# Final Prototype (Group Project)

A web-based prototype created as part of a group ICT project.

---

## About the Project  
This project is a final prototype built from a Figma design. The goal of the project is to turn a UI/UX design into a working web application.

The original design can be found here:

## Running the Project  

### Install dependencies  
```
npm install
```

### Run development server  
```
npm run dev
```

After running the project, open your browser at:  
```
http://localhost:3000
```

---

## Project Structure  
- `/src` → main source code  
- `/components` → reusable UI components  
- `/pages` → different pages of the application  
- `/assets` → images and static files  

---

## Development Process  
We worked in a group using a Git workflow. We made regular commits while building different parts of the project. This helped us track changes and combine work from different team members.
